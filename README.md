# SEC-Beautify

All publicly traded companies are required to file reports with the SEC. The SEC does a great job of making all these reports available at [www.sec.gov](https://www.sec.gov); however, the webpage for viewing these reports is not well designed. The text is way too long, making it difficult to read. Tables are similarly too long. This browser extension solves this problem by limiting the width of the text and tables on just the SEC pages for viewing reports. This extension also removes the page numbers, page section breaks, and red outlines on the website, making for an easier web-based reading experience.

Note: The SEC does not make some of the HTML code easy to select and remove. For example, to remove the page numbers, this plugin had to select and remove all elements with a height of 42.75pt. I make no guarantee that this extension might not remove critical information from the website. By using this plugin, you agree to always double check the SEC's website without the plugin enabled before making any investment decisions.
